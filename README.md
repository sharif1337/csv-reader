# CSV Reader
> You can Read Easily CSV file
### Show Columns from file
```
python3 csv-reader.py -sc
```
### Extract Data from Columns
```
python3 csv-reader.py -f filename.csv -c column1 column2 "unknwon column" -o output.txt
```
